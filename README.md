# NestJS RESTful API Designs Playground

---

> *This project is my practice playground for fast and effective designing RESTful API with NestJS.*

---

## Installation

```bash
yarn install
```

## Running the app

Development:

```bash
yarn run start
```

Watch mode:

```bash
yarn run start:dev
```

Production mode:

```bash
yarn run start:prod
```

---
